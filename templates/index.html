House Price Prediction App

Hopred is House Price Prediction App. The main aim of the project is to build
an app that can predict house prices depending on various perspective and the
features of the house that one wants to buy for example; how many bedrooms
the buyer wants or does the buyer wants an air-conditioning, basement etc.
The project will be based on a housing dataset, in which the dataset will be
analyzed and predicted using various machine learning models and python
programming language.
